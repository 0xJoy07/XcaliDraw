# XcaliDraw

An interactive whiteboarding application with a rich set of drawing tools, custom styling, and AI-powered layout organization.

## Quick Start
1. `npm install`
2. `npm run dev`

## Tech Stack
- React + TypeScript + Vite
- Zustand (State Management)
- Rough.js (Hand-drawn styled shapes)
- Perfect-Freehand (Smooth freehand drawing)
- RBush (Spatial indexing for hit-testing and snapping)
- IndexedDB via idb-keyval (Image persistence)
